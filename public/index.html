Instructions

This is a tool created to maintain save files for Exponential Idle. Saves are stored as base64 encoded string, which can be obtained from the game by going to Settings, Import/Export and clicking on Export to clipboard. This generates the save string and copies it to the clipboard.

Click on "Add/Edit Save" and paste in the save string generated from the app in the text area labeled export. Note down and keep the following data points handy, to fill in the "Add a new Save" dialog. This will help you identify saves later on:

  • f(t)
  • Lifetime f(t)
  • μ
  • ψ
  • σ

Note that all the above fields are optional, but filling them is recommended. If a save with the same name exists, it's values will be overwritten with the new ones

Click on Existing Saves to list all previously saved Saves. The icon can be used to copy that save to the clipboard. The icon can be used to delete a particular save. Note that this action cannot be reversed.
